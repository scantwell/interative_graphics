This assignment consists of one program.
        - a7

Features:
	- Reads in and stores control points.
	- Tesselates patch
	- Smooth-shading
	- Draws X, Y, Z axes
	- Select control points
	- Move control points
	- Updating patch triangles
	- Modify triangle resolution
	- Implements Phong algorithm.
	- Camera Rotates around the object when toggled
	- Camera can be adjusted by six keys (see a4 print out for details)

How to Run:

 make
        To compile all of source files

 make run
        To run the program.

 make clean
        Cleans all artifacts.
