This assignment consists of two different programs.
        - red_triangle
        - extra_exmample

red_triangle displays four different shapes which include an elipse, triangle, circle, and square. The circle is not all one color a
lthough it does by appearance. In fact, the angle is used to compute the color value.

extra_example is a snowman with face and arms.

How to Run:

 make
        To compile all of source files

 make run
        To run the red_triangle program.

make run2
        To run the extra_example program.

 make clean
        Cleans all artifacts.
